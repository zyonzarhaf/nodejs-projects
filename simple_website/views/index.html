why learn node.js ?

Node.js is an open-source, server-side runtime environment. It allows you to run JavaScript on the server. Node.js is commonly used for programming web servers, i.e., server-side JavaScript.

Node.js is cross-platform, so you can run it on Windows, macOS, and Linux, and it uses the JavaScript programming language. It's a powerful runtime environment that has been ranked the technology most commonly used by developers.

Node.js allows you to build highly-scalable, fast, and secure RESTful APIs with Node, Express, and MongoDB. It's a great tool for building modern, modular JS applications.
